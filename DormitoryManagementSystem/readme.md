## 宿舍管理系统

### 1.为什么要做

1. 暑假实验室考核项目
2. 数据结构课设之一

### 2.要求

1. 采用交互工作方式为宿舍管理人员编写一个宿舍管理查询软件。（qt实现）
2. 增删改查（sqlite实现）
3. 文件操作（.db文件）
4. 搜索 排序 信息查询

### 3.技术

1. C++(QT)
2. sqlite
